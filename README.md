# Budżet domowy PyQt

Projekt pomocniczy do zaliczenia z programowania w Pythonie.

Aplikacja desktopowa służąca do zarządzania domowym budżetem. Program pozwala wczytać dane o wydatkach z pliku Excel, wyświetlić je w tabeli oraz wygenerować wykres przedstawiający sumę wydatków w poszczególnych kategoriach.

## Wymagane biblioteki

Program korzysta z bibliotek:

- PyQt5
- pandas
- matplotlib
- openpyxl

Instalacja bibliotek:

```bash
pip install -r requirements.txt
